## HOW TO APPROACH THIS TASK

1. First, identify where global styles/CSS variables are defined (likely a globals.css, index.css, or theme file). Update all color and font variables there first.
2. Then go screen by screen in this order: Auth → Classrooms → Sidebar → Header → Overview → Heatmap → Mastery → Homework → Students.
3. For each screen, find the component file, update className styles or CSS.
4. If the project uses Tailwind: replace utility classes with the new palette. If it uses CSS modules or styled-components: update the style definitions directly.
5. After each screen, verify nothing is broken functionally.
6. Do a final pass to ensure font imports are loading, all cyan/blue has been replaced, and the grid background is showing correctly on all pages.
